## 一、项目概述

财务工作台 v2.0 是由**一键财报分析 v1.1**和**财务工具箱 v1.5**整合而成的统一财务 software 平台，采用**前端统一入口 + 后端模块化**架构，保留全部 25 个页面功能，新增 3 个功能模块（智能透视分析、增强功能、帮助中心）。

### 核心特性

| 特性 | 说明 |
|------|------|
| 统一入口 | 单 `app.py` 集成认证和导航，侧边栏 5 大模块切换 |
| 模块化设计 | 按工作场景分为 5 大模块：日常核算、财务分析、数据工厂、决策支持、办公工具 |
| 缓存机制 | `@st.cache_resource`、`@st.cache_data` 多级缓存 |
| 性能优化 | 分页查询、数据库索引（16 个）、批量操作优化 |
| 数据准确性 | 行业对标使用 2025-2026 年权威统计数据 |
