## 快速启动

### 首次运行

双击软件目录下的 `一键启动.bat`，自动完成初始化。

### 后续启动

**方式一**（推荐）：双击 `启动.py`

**方式二**：双击 `一键启动.bat`

### Linux/Mac

```bash
cd 一键财报分析
python3 配置文件.py  # 首次运行
python3 启动.py       # 后续启动
```

## 访问地址

启动后浏览器打开：`http://localhost:8501`

## 文件说明

| 文件 | 说明 |
|------|------|
| `配置文件.py` | 首次运行初始化（创建虚拟环境、安装依赖） |
| `启动.py` | 快速启动应用（跳过安装） |
| `一键启动.bat` | 自动判断是否需要初始化 |

## 停止服务

按 `Ctrl+C`
