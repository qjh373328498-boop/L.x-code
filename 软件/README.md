## 快速启动

### Windows

双击根目录的 `一键启动.bat`，选择软件后：
- 首次：自动创建虚拟环境、安装依赖
- 后续：直接启动应用（秒启动）

### Linux/Mac

```bash
# 首次运行
python3 配置文件.py

# 后续启动
python3 启动.py
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `配置文件.py` | 统一配置入口，选择软件后创建虚拟环境 |
| `启动.py` | 统一启动入口，选择软件后快速启动 |
| `一键启动.bat` | 自动判断首次/后续运行 |
