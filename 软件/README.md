# 软件集

包含 3 个 Streamlit 应用：

1. **🏆 学创杯辅助软件 v2.2** - 14 个功能模块
2. **💼 财务工具箱 v1.5** - 15 个功能模块

## 快速启动

### Windows
双击运行 `一键启动.bat`

### Linux/Mac
```bash
cd 一键财报分析 && ./一键启动.sh
```

## 访问地址
启动后浏览器打开：`http://localhost:8501`

## 停止服务
按 `Ctrl+C`
