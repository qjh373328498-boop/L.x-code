## 🎯 五档评价标准说明

### 定义

| 档次 | 分位值 | 说明 |
|------|--------|------|
| **优秀** | 前 10% | 行业标杆水平 |
| **良好** | 前 25% | 优于四分之三企业 |
| **平均** | 中位数 | 行业平均水平 |
| **较低** | 后 25% | 低于四分之三企业 |
| **较差** | 后 10% | 行业尾部水平 |

### 使用场景

```python
from utils.industry import compare_with_industry

# 对比企业财务指标与行业水平
result = compare_with_industry('制造业', '流动比率', 1.8)

# 输出:
{
    "value": 1.8,
    "excellent": 2.5,      # 优秀值 (前 10%)
    "good": 2.0,           # 良好值 (前 25%)
    "average": 1.5,        # 平均值 (中位数)
    "low": 1.0,            # 较低值 (后 25%)
    "poor": 1.0,           # 较差值 (后 10%)
    "rating": "平均",      # 评级
    "comment": "处于行业平均水平",
    "gap_to_excellent": -0.7,  # 与优秀值差距
    "gap_to_average": 0.3      # 与平均值差距
}
```

## 📝 使用建议

### 1. 行业选择优先级
1. **优先使用**: 国务院国资委标准（最权威）
2. **参考使用**: 上市公司协会统计（代表性强）
3. **辅助参考**: 专业机构标准（教学重点）

### 2. 评级使用
- **优秀**: 激励标杆
- **良好**: 合理目标
- **平均**: 底线要求
- **较低/较差**: 预警信号

### 3. 注意事项
- 考虑行业特殊性（如金融业高负债）
- 结合企业发展阶段
- 参考历史趋势
- 关注季节性因素
