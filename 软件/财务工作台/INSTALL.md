# 财务工作台 v2.0 - 快速安装指南

## 🚀 一键启动（推荐）

### Windows 系统

双击运行 **一键启动.bat**

### Mac / Linux 系统

```bash
chmod +x 一键启动.sh
./一键启动.sh
```

---

## 📦 手动安装

### 1. 环境要求

- Python 3.8 或更高版本
- 内存：2GB+
- 存储：100MB+

### 2. 安装依赖

**方式一：使用官方源**
```bash
pip install -r requirements.txt
```

**方式二：使用清华镜像源（国内推荐）**
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**方式三：使用阿里镜像源**
```bash
pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
```

### 3. 启动应用

```bash
streamlit run app.py --server.port 8502 --server.address 0.0.0.0
```

### 4. 访问系统

浏览器打开：**http://localhost:8502**

## 🛠️ 常见问题

### 1. 无法启动

**检查 Python 版本**
```bash
python --version
```

**重新安装依赖**
```bash
pip install --upgrade -r requirements.txt
```

### 2. 端口被占用

**修改端口**
```bash
streamlit run app.py --server.port 8503
```

### 3. 数据丢失

**检查数据库文件**
```
data/finance.db
```

**从备份恢复**
```bash
cp data/finance_backup_*.db data/finance.db
```

### 4. 浏览器无法访问

**检查防火墙设置**
- Windows：允许 Python 通过防火墙
- Mac：系统偏好设置 → 安全性与隐私 → 防火墙

**使用本地访问**
```
http://127.0.0.1:8502
```
