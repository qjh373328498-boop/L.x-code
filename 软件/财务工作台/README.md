# 财务工作台 v2.0

> 整合财务工具箱 v1.5 和 FinCopilot v1.0 的统一财务工作平台

## 📦 项目结构

```
财务工作台/
├── app.py                          # 统一入口
├── requirements.txt                # 依赖清单
├── .streamlit/
│   └── config.toml                 # Streamlit 配置
├── core/                           # 核心算法库
│   ├── extractor.py                # 文档提取
│   ├── calculator.py               # 金融计算
│   ├── cleaner.py                  # 数据清洗
│   ├── compliance.py               # 合规检查
│   ├── matcher.py                  # 智能匹配
│   ├── reporter.py                 # 报表生成
│   └── desensitizer.py             # 数据脱敏
├── utils/                          # 工具模块
│   ├── database.py                 # 数据库连接
│   ├── formatters.py               # 格式化工具
│   ├── ratios.py                   # 财务比率
│   ├── industry.py                 # 行业标准
│   ├── export_helper.py            # 导出工具
│   ├── history_manager.py          # 历史记录
│   └── theme_switcher.py           # 主题切换
├── pages/                          # 功能页面
│   ├── 10_日常核算/                # 6 个页面
│   ├── 20_财务分析/                # 5 个页面
│   ├── 30_数据工厂/                # 4 个页面
│   ├── 40_决策支持/                # 3 个页面
│   └── 50_办公工具/                # 4 个页面
├── data/                           # 数据库目录
└── templates/                      # 模板文件
```

## 🚀 快速开始

### 1. 安装依赖

```bash
cd 财务工作台
pip install -r requirements.txt
```

### 2. 启动应用

```bash
streamlit run app.py
```

### 3. 访问系统

浏览器打开 `http://localhost:8501`

## 📊 功能模块

### 📝 日常核算（6 个页面）
1. **发票管理** - 发票录入、查询、认证
2. **银行对账** - 银行流水与日记账匹配
3. **凭证录入** - 会计凭证填写与保存
4. **科目余额表** - 科目汇总与明细查询
5. **应收应付管理** - 往来账款跟踪
6. **纳税申报** - 税务报表生成

### 📊 财务分析（5 个页面）
1. **财务比率分析** - 偿债能力、营运能力、盈利能力
2. **杜邦分析** - ROE 拆解与可视化
3. **行业对标** - 与行业平均值对比
4. **资金诊断** - 现金流健康度评估
5. **预算分析** - 预算执行情况跟踪

### 🏭 数据工厂（4 个页面）
1. **文档解析** - PDF/图片 OCR 识别
2. **批量解析** - 批量处理文档
3. **数据治理** - 数据清洗与脱敏
4. **合规风控** - 合规性检查

### 💼 决策支持（3 个页面）
1. **金融测算** - 货币时间价值计算
2. **本量利分析** - 盈亏平衡点分析
3. **报表美化** - 财务报告格式化

### 🧰 办公工具（4 个页面）
1. **财务日历** - 重要日期提醒
2. **快捷工具箱** - 常用小工具集合
3. **模板中心** - 报表模板下载
4. **历史记录** - 操作历史查询

## 🛠️ 开发说明

### 数据库
系统使用 SQLite 数据库，数据文件位于 `data/finance_workbench.db`

### 导入模块
页面中如需使用工具函数，请使用以下导入方式：

```python
from utils.database import get_connection
from core.calculator import calculate_pv, calculate_fv
from core.extractor import extract_from_pdf
```

### 主题切换
系统已集成统一的主题切换功能，在侧边栏可随时切换明/暗主题。
