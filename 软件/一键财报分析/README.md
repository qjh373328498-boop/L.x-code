# 📊 一键财报分析

专业的财务报表自动化分析工具，内置 8 大行业标准值对比，6 大维度全面分析。

## ✨ 核心功能

- **智能解析**：自动提取 Excel/PDF 财报中的关键数据
- **行业对标**：与制造业、科技、零售、房地产等 8 大行业标准值对比
- **六大维度分析**：
  - 盈利能力（毛利率、净利率、ROE 等）
  - 偿债能力（流动比率、资产负债率等）
  - 运营效率（存货周转率、应收账款周转率等）
  - 现金流分析（经营现金流、盈利现金比率等）
  - 成长性分析（营收增长、利润增长等）
  - 风险预警（自动识别财务异常信号）
- **智能诊断报告**：一键生成综合评分和专业分析报告

## 🚀 快速开始

### 1. 安装依赖

```bash
cd 软件/一键财报分析
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. 启动应用

```bash
# 方式一：直接启动
streamlit run app.py

# 方式二：使用启动脚本（开发中）
一键启动.bat
```

### 3. 访问应用

浏览器打开：http://localhost:8501

## 📁 支持的财报格式

### Excel 格式（推荐）
- `.xlsx` 或 `.xls`
- Sheet 名称建议包含：
  - 资产负债表："资产"、"balance"
  - 利润表："利润"、"income"、"损益"
  - 现金流量表："现金流"、"cash"

### PDF 格式
- 支持标准财务报表 PDF
- 自动提取关键财务数据

## 🏭 支持的行业

| 行业 | 关键指标示例 |
|------|-------------|
| 制造业 | 固定资产周转率、存货周转率 (4-12 次)、毛利率 (20-40%) |
| 科技/互联网 | 毛利率 (60-80%)、研发投入占比 (10-25%)、ROE (15-35%) |
| 零售业 | 存货周转率 (20-30 次)、毛利率 (20-30%)、总资产周转率 (2-3 次) |
| 房地产 | 存货周转率 (0.3-0.8)、资产负债率 (60-85%)、土储倍数 (3-5 年) |
| 金融业 | 拨备覆盖率 (150-300%)、资本充足率 (10-15%)、ROE (10-20%) |
| 服务业 | 人均产出、毛利率 (30-50%)、ROE (15-30%) |
| 医药行业 | 毛利率 (60-90%)、研发费用率 (10-25%)、ROE (15-30%) |
| 消费品 | 毛利率 (40-70%)、销售费用率 (15-35%)、存货周转率 (5-15 次) |

## ⚠️ 注意事项

1. **数据准确性**
   - 系统基于上传的财报数据进行计算
   - 确保上传的是经审计的正式财报
   - PDF 解析可能存在误差，建议优先使用 Excel

2. **行业识别**
   - 行业识别基于公司名称和业务范围关键词匹配
   - 如识别不准确，可手动切换行业

3. **数据隐私**
   - 所有数据仅存储在本地
   - 不会上传到任何服务器
