### Windows 用户

双击运行：
```

```

会显示交互式菜单，可以选择要启动的软件。

#### 方式二：直接启动单个软件

进入对应软件目录，双击：
- `学创杯辅助软件/一键启动.bat`
- `财务工具箱/一键启动.bat`
- `一键财报分析/一键启动.bat`

### Linux/Mac 用户

```bash
cd 软件/一键财报分析
chmod +x 一键启动.sh
./一键启动.sh
```

## 常见问题

### ❌ 启动失败：未检测到 Python

**解决方法**：
1. 下载安装 Python 3.8 或更高版本
2. 下载地址：https://www.python.org/downloads/
3. 安装时勾选 "Add Python to PATH"

### ❌ 端口被占用

**错误信息**：`Address already in use`

**解决方法**：
1. 关闭已运行的应用
2. 或修改启动命令中的端口号：
   ```bash
   streamlit run app.py --server.port 8502
   ```

### ❌ 虚拟环境创建失败

**解决方法**：
1. 手动删除软件目录下的 `venv` 文件夹
2. 重新运行启动脚本

### ❌ 依赖安装失败

**解决方法**：
1. 检查网络连接
2. 手动安装依赖：
   ```bash
   cd 软件/xxx
   python -m pip install -r requirements.txt
   ```

## 访问应用

启动成功后，在浏览器中打开：
```
http://localhost:8501
```

## 停止应用

按 `Ctrl+C` 可停止服务

## 技术支持

如遇到问题，请检查：
1. Python 版本（3.8+）
2. 网络连接（需要下载依赖）
3. 端口占用情况
4. 查看错误日志
